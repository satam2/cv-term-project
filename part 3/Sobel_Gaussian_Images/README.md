# Sobel Edge Detector — Performance wwith Gaussian Filter

Applying the Sobel edge detector with Gaussian, the results had a softer image. The noise reduced because the weaker lines were blurred away. For Sobel the image was convolved with a gaussian 5x5 filter. Then convolved with a horizontal and vertical gradient. Magnitude was calculated for every pixel and normalized. With the Guassian blur, noise was reduced by focusing on the stronger lines. The edge thickness show for lines in the image that were noticeable.
