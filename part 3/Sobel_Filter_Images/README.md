# Sobel Edge Detector — Performance Under Noisy Conditions

When applied directly to the noisy image without Gaussian pre-filtering, the Sobel operator produces thick, granular edges with substantial speckle scattered across otherwise uniform regions, since its 3x3 gradient kernels amplify high-frequency intensity fluctuations introduced by the noise. Edge quality is degraded: true object boundaries are still detectable but appear noisy and surrounded by spurious gradient responses, making the result visually cluttered. Continuity along strong edges is generally preserved because Sobel reports gradient magnitude everywhere rather than thinning, but weaker edges blend into the noise floor and lose definition. Noise sensitivity is high — the operator has no built-in smoothing or thresholding stage, so every local intensity jump (signal or noise) is treated as a candidate edge. This confirms that Sobel requires a smoothing pre-step (e.g., Gaussian blur) for reliable use on noisy inputs.
